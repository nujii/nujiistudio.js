# Nujii Studio
JavaScript assets for Nujii Studio Digital Audio Workstation

## Getting started

Clone the repo and run `npm install` & `brunch build`.
See more info on the [official site](http://brunch.io)

## Usage

    npm start

then visit `localhost:3333`
